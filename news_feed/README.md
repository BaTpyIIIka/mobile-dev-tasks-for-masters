#### This project is the best students news feed Android app. 

### Key features:
- Auth via email
- Data caching
- Open full article in a browser

### Used technologies:
- Kotlin v1.3
- Firebase for authentication
- Firestore as datastorage
- Android material

### App requirements:
- minSdkVersion 21
- targetSdkVersion 29
